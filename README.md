# Chat Terminal

## Requirements

- Python 3.11 or higher
- Poetry

## Install

1. Clone the repository

   ```bash
   git clone https://github.com/wise-ntust/chat-terminal.git
   ```

2. Install dependencies

   ```bash
   poetry install
   ```

3. Build the project

   ```bash
   poetry build
   ```

4. Install the package

   ```bash
   pip install dist/chat_terminal-0.1.0-py3-none-any.whl
   ```

## Run

```bash
chat-terminal
```
